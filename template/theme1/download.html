1.如何安装游戏大厅。

　A.首页中点击‘立即体验下载版’，请以下列方法择一安装。

　　 i.点击‘ INSTALL NOW ’将会进行安装程序。

　　ii.点击‘立即下载’，将压缩档解压缩之后，执行安装程序，将会进行安装程序。

 

2.游戏大厅安装失败。

　A.请检查网路是否正常，以及档案是否正常下载，或是使用续传软体下载完整安装档，执行安装程序进行完整安装。

　B.如果还是失败，请检查系统中，是否有资料夹或是其他软体占用。

　　i.举例，该游戏大厅名称为 hall，或是 hall.zip（与下载下来的档案名称一致）。

　　 32位作业系统请删除：‘ C:\Program Files\hall\ ’资料夹之后重新安装大厅。

　　 64位作业系统请删除：‘ C:\Program Files (x86)\hall\ ’资料夹之后重新安装大厅。

 

3.如何下载并且安装游戏。

　A.安装完游戏大厅之后，请以下方法安装。

　　i.在尚未下载游戏的时候，会在游戏图示上呈现‘尚未安装’，请点击游戏图示之后，游戏大厅将会自动下载并且安装游戏，待安装完毕即可进入游戏。

 

4.点选游戏下载无反应，无法下载游戏档案。

　A.请先检查网路，是否断线或是被防火墙挡住。

　B.可能为 Vista 或是 win7 的权限问题，请以下列方法择一解决。

　 　 i.将UAC关闭。

　 　ii.下载版捷径按滑鼠右键，点选‘以系统管理员身份执行’。

　　iii.下载版捷径按滑鼠右键，点选‘内容’、‘相容性’、‘特殊权限等级’，将‘以系统管理员的身份执行此程式’选项打勾。

1.什么是AIR runtime。

 

 

　A. Adobe AIR（AIR＝Adobe Integrated Runtime），开发代号为Apollo，是一个跨作业系统运行环境，用来建造RIA，使用的Flash、Flex、HTML与AJAX，可能部署为桌面应用程式。AIR是Adobe针对网路与桌面应用的结合所开发出来的技术，可以不必经由浏览器而对网路上的云端程式做控制，也由于这是的Adobe所开发的技术，因此能很顺利的与的Adobe旗下的Photoshop、Flash、Fireworks等应用程式来进行开发。

 

2.为什么需要AIR runtime。

　A.Runtime是一种介于编译器与直译器的执行方式，指一种把半编译的执行码在某台机器上面执行。如的Java与.Net也是分别在JAVA runtime、以及.Net runtime底下执行。

 

3. Vista、Windows7 底下的权限问题。

　A.因为 Vista 以及 Windows7 对于系统资料夹的限制较为严格，如 windows、program file 资料夹，以致于在一般使用者之下，下载版从网路上下载的游戏程式无法写入到硬碟中。

　B.为了解决这个问题，必须要以系统管理员身份执行下载版游戏，有下列几种方式：

　　i.将 UAC 关闭，在 win7 中搜寻‘ uac ’，即可显示设定视窗。

 

 

　　ii. 对游戏大厅捷径点击滑鼠右键，选择‘以系统管理员身份执行’。

 

 

　　iii.对游戏大厅捷径点击滑鼠右键，选择‘内容’、‘相容性’下方的‘特殊权限等级’，将‘以系统管理员的身份执行此程式’勾选。

1.什么情况下需要看此文件。

　A.当更新画面维持很久没有变动时，有可能是因为 UAC 的关系。

 

2.什么是UAC。

　A.使用者帐户控制（UAC, User Account Control）是 Windows Vista 中的新安全技术，通过限制应用软体而改进的Windows作业系统的安全性。Windows 7 中保持了这个功能，但是略有改变。

 

3. UAC 有什么作用。

　A.当某些动作会影响系统的安全及稳定性或影响运作时，UAC 便会弹出讯息，在执行前要求管理员的帐户及密码，除了该对话框之外，荧幕其他部份都会变暗，用户不能进行其他操作。若果用户的群组是“系统管理员”，则只需在对话方块中选择“允许”或“不允许”，若果用户的群组是“标准使用者” ，则需请求管理员的授权及密码。

　B.透过启动前先确认这些动作，UAC 可协助防止恶意软体（malware）与间谍软体未经授权就进行安装或变更电脑。正因如此，在Windows 系统中，UAC 预设为开启的。

 

4.对下载版有什么影响。

　A.下载版会在本机端放置游戏档案，这样使用者就不需要每一次玩游戏都需要从网路上下载，当如果本机中有最新版本的游戏，则只需要从本机端载入，以减少载入时间。 因为下载版会依照线上的定义档，来确保本机上的游戏是最新的状态，所以必须要将线上最新的档案下载到本机中，因为Vista或是win7 UAC 的保护，以致于无法正确写入到硬碟中。
